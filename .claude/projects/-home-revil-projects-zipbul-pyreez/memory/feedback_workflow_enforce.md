---
name: workflow-always-enforce
description: Always follow workflow.md flow — Classify with human approval before any work
type: feedback
---

모든 작업에 workflow.md의 플로우를 적용한다. 대화 속도와 무관하게 Classify → human approval → 실행 순서를 반드시 거친다.

**Why:** 사용자의 빠른 질문/지시에 반응적으로 따라가느라 Classify 단계를 생략하고 승인 없이 작업을 진행함. 스킬 파일 정리, REMAIN_2.md 작성, 엔진 버그 수정 등을 플로우 없이 진행.

**How to apply:** 새로운 작업 단위가 시작될 때마다 (1) 어떤 플로우인지 Classify하고 (2) 사용자 승인을 받은 후에만 진행. "빠르게 하는 것"보다 "플로우를 지키는 것"이 우선.
